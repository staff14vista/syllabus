Lesson Plan - Word

SL NO Menu/Module Session Detailed Sub-Topics
1 File Menu 1.1 New: Blank document, templates, online templates
1.2 Open: Local files, recent documents, OneDrive integration(if possible)
1.3 Save / Save As: DOCX, PDF, Word 97-2003, templates, etc.
1.4 Print: Print preview, select printer, page setup, print options
1.5 Share: Email, cloud sharing, PDF/XPS, invite people(if possible)
1.6 Export: Create PDF/XPS, change file type, package document
1.7 Info: Protect Document (read-only, restrict editing), Version History, Document Properties
1.8 Options: General, Display, Proofing, Save, Language, Advanced, Customize Ribbon
2 Home Tab 2.1 Clipboard: Cut, Copy, Paste, Paste Special, Format Painter
2.2 Font: Font type, size, bold, italic, underline, strikethrough, subscript, superscript, text effects, text color, highlight, clear formatting
2.3 Advanced Font Options: Character spacing, font scaling, position
2.4 Paragraph: Alignment (left, center, right, justify), Indentation (left, right, special), Line spacing, Paragraph spacing, Bullets, Numbering, Multilevel lists, Borders, Shading, Paragraph settings dialog box
2.5 Styles: Apply quick styles, modify existing styles, create new styles, manage style sets, apply theme styles
2.6 Editing: Find, Replace, Select All, Select Objects
2.7 Advanced Features: AutoCorrect, AutoText, Quick Parts, Themes
3 Insert Tab 3.1 Pages: Cover Page, Blank Page, Page Break
3.2 Tables: Insert Table (grid, dialog box), Draw Table, Convert Text to Table, Quick Tables, Table Styles, Table Design Options, Cell Margins, Split/Merge Cells, Sort, Formulas
Complete hands-on knowledge on Table Design and Format Tab
3.3 Illustrations: Pictures (local/online) along with Picture Format Tab,
Shapes along with Shape Format Tab,
SmartArt along with SmartArt Design and Format Tab,
Charts along Chart Design and Format Tab, Screenshot, 3D Models.
3.4 Links: Hyperlink, Bookmark, Cross-reference, Link to external documents
3.5 Header/Footer: Predefined styles, Edit, Remove, Different First Page, Odd/Even Pages
3.6 Page Number: Top, Bottom, Format, Remove
3.7 Text: Text Box, WordArt, Drop Cap, Signature Line, Date & Time, Object (Excel, PDF)
3.8 Symbols: Symbol dialog box, Equation
4 Design Tab 4.1 Document Formatting: Themes, Colors, Fonts, Paragraph Spacing, Effects, Set as Default
4.2 Page Background: Watermark, Page Color, Page Borders
4.3 Document Formatting Options: Change Style Set, Customize Formatting
5 Layout/Page Layout 5.1 Page Setup: Margins, Orientation, Size, Columns, Breaks, Line Numbers, Hyphenation
5.2 Paragraph: Indent (left/right), Spacing (before/after)
5.3 Arrange: Position, Wrap Text, Bring Forward, Send Backward, Align, Group, Rotate
6 References Tab 6.1 Table of Contents: Automatic, Manual, Update
6.2 Footnotes / Endnotes: Insert, Convert, Next/Previous, Show Notes
6.3 Citations & Bibliography: Add new source, Manage sources, Style (APA, MLA, Chicago), Insert bibliography
6.4 Captions / Index / Table of Authorities: Insert caption, cross-reference, update index, mark entries
7 Mailings Tab 7.1 Create: Envelopes, Labels
7.2 Start Mail Merge: Letters, Email, Labels, Directory
7.3 Select Recipients: Type new list, Use existing list (From Excel), Choose from Outlook contacts(if Possible)
7.4 Write & Insert Fields: Insert merge fields, Rules, Address block, Greeting line
7.5 Advanced Mail Merge: Conditional fields (IF/ELSE), Merge to email with attachments
7.6 Preview & Finish: Preview results, Complete merge, Edit individual documents
8 Review Tab 8.1 Proofing: Spelling & Grammar, Thesaurus, Research
8.2 Language: Translate, Language preferences
8.3 Comments: New Comment, Delete, Previous/Next, Show/Hide
8.4 Track Changes: Track, Accept/Reject changes, Display options
8.5 Compare & Protect: Compare documents, Combine, Restrict Editing
8.6 Accessibility Checker
9 View Tab 9.1 Document Views: Read Mode, Print Layout, Web Layout, Outline, Draft
9.2 Show/Hide: Ruler, Gridlines, Navigation Pane
9.3 Zoom: Zoom In/Out, 100%, One Page, Two Pages, Page Width
9.4 Window: New Window, Arrange All, Split, View Side by Side, Switch Windows
9.5 Macros: Record Macro, Assign Shortcut, Run Macro
10 QnA
